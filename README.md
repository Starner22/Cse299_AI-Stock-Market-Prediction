# ai-stock-market-prediction
This project aims to predict stock market trends and assess investment risks using Artificial Intelligence (AI) and Machine Learning.   It combines historical data analysis, feature engineering, and predictive modeling to deliver data-driven insights for smarter financial decisions.  
Models used:
1. Linear Regression
2. Ridge Regression
3. Prophet
4. Support Vector Regression (SVR)
5. Long Stort Term Memory (LSTM)
6. Gated Recurrent Unit (GRU
7. Neural Basis Expansion Analysis for Interpretable Time Series (N-Beats)
8. Temporal Fusion Transformer (TFT)
